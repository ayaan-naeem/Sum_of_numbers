This is a basic console application where you can enter two numbers, and it will calculate and display their sum. It's a simple program, but it marks the beginning of my journey into mastering C# and building more advanced programs in the future.

## Features

Takes user input for two numbers.

Calculates the sum of the numbers.

Uses Console.Write and Console.WriteLine for input/output formatting.

Demonstrates basic variable usage, parsing input, and string interpolation in C#.

## How to Use

Clone this repository.

Open the project in Visual Studio (or your preferred C# IDE).

Run the program.

Follow the prompts to enter two numbers.

See the calculated sum printed on the console.

## Example:

Enter the first number: 5
Enter the second number: 7
The sum will be: 12
